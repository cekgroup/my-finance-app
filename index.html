Google Sheets sync
Step 1 — Create your spreadsheet
Go to sheets.google.com, create a new sheet, name it My Finance Tracker.
Step 2 — Open Apps Script
In your sheet, click Extensions → Apps Script. Delete all existing code and paste the script from the file google-apps-script.js included with this app.
Step 3 — Deploy as web app
Click Deploy → New deployment. Set Execute as: Me. Who has access: Anyone. Click Deploy and copy the URL.
Step 4 — Paste your URL below
Paste the deployment URL here and tap Save.
