Two-way cloud sync
Once connected, every change you make here automatically syncs to the cloud within a couple seconds, and any device with the same URL pulls those changes automatically — when you open the app, every 30 seconds while it's open, and whenever you switch back to it.
Step 1 — Create your spreadsheet
Go to sheets.google.com, create a new sheet named My Finance Tracker.
Step 2 — Open Apps Script
In your sheet: Extensions → Apps Script. Delete all code, paste the script from google-apps-script.js.
Step 3 — Deploy as web app
Deploy → New deployment → Web app. Execute as: Me. Who has access: Anyone. Copy the URL.
Step 4 — Paste URL below (do this on EVERY device)
Use the exact same URL on your phone and your laptop so they sync to the same place.
